### 사용한 라이브러리

 - ☑️Alamofire :
	 - HTTP 통신을 위해 사용
 - ☑️SwiftyJSON
	 - JSON 객체를 파싱하기 위해 사용
 - ☑️Firebase
	 - RealTime DB를 활용한 채팅 기능을 구현하기 위해 사용
 - ☑️ObjectMapper 
	 - JSON 객체로 변환하기 위해 사용
 - ☑️SideMenu 
	 - 햄버거 사이드바 메뉴 구현하기 위해 사용
 - ☑️GIFImageVIew 
	 - GIF 이미지 띄위기 위해 사용
 - ☑️RealmSwift 
	 - 최근 검색어 목록을 저장하기 위해서 사용
